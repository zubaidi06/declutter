# Declutter
C#.net CLI tool to help you organize messy folders

You decide on the ruleset. Declutter will organize your filed into folders based on your preference. There is no recursion yet.

### How to run

1. clone
2. all comands start with "dotnet run"
3. to see a list of commands, run "dotnet run help"
4. tell Declutter how you want your files organized using the "rule" commands
5. Pick the folder to organize with "dotnet run organize <path_to_folder>
